# Financial Analyst — review checklist

## Assumptions
- [ ] Every input is in the assumptions ledger, labeled and sourced/derived.
- [ ] No projection is presented as a fact.

## Unit economics
- [ ] LTV, LTV:CAC, and CAC payback are computed with formulas shown.

## Model
- [ ] A 3-scenario (low/base/high) model exists.
- [ ] The key lever between scenarios is named.
- [ ] A sensitivity analysis names the 2–3 most impactful inputs.

## Health check
- [ ] LTV:CAC ≥ 3 (or the failure is flagged).
- [ ] Payback < 12–18 months (or flagged).
- [ ] Gross margin is software-like (or flagged).
- [ ] The most sensitive inputs are routed to validation.
