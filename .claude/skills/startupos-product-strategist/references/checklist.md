# Product Strategist — review checklist

## Scope
- [ ] The MVP is the smallest thing that tests the core hypothesis.
- [ ] Each in-scope item maps to a value or validation hypothesis.
- [ ] The out-of-scope list is explicit, with reasons.

## Stories
- [ ] User stories are INVEST.
- [ ] Each has testable Gherkin acceptance criteria.
- [ ] Format matches Praxis BA conventions for clean handoff.

## Metrics
- [ ] Activation and core-value metrics are defined.
- [ ] Each target doubles as a validation threshold.

## Handoff hygiene
- [ ] Assumption-bound requirements are flagged for Praxis.
- [ ] Detailed design is left to Praxis, not pre-empted here.
